# My-Assignment2
This is my assignment 2 which is an app that is a game that helps users distinguish between genuine productivity tips 
and urban myths that have gone viral with flashcards.
This app provides a "Life Hack or Urban Myth" flashcard game 
to help users test their common sense and learn useful (and safe) real-world shortcuts.
The app present questions, loop through each hack 
and provide a total score based on the user's ability to spot a myth.
